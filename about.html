strauh.al is a repository of almost everything I (ernest.strauh.al) have consumed through the internet since 2015/2016ish. My artworks and music live on studio.strauh.al.

This website is forever a work in progress, as there are currently 1,432 images in my downloads folder that I need to sort, along with other folders on my hard drive that still have to be uploaded to this website. My bookmarks in my web browser are also getting out of hand and need to be sorted soon so that they can be uploaded to the "knowledge_base" portion of this website.

This site runs on GitHub pages, and can be viewed on the GitHub website here: github.com/strauhal/strauh.al. The design language of strauh.al is purposely bleak as I've tried to keep the HTML and CSS code to the bare minimum. As of January 2023, there is no Javascript code on this website.

Talking to people in real life is often tedious and slow, and I much prefer this method of disseminating information about myself in the form of this website, where I can hyperlink thoughts, ideas, and influences.
